Menu Driven Python Program

Project is a basic menu driven program written in Python. User can select one of the three options in the menu and perform their task.

Program

Project is a basic menu driven program written in Python. User can select one of the three options in the menu and perform their task.

Language

The program is written in Python.

Option 1 - Calculator

User can input two numbers and perform the basic mathematical operations like add, subtract, multiply and divide.

Option 2 - Area Of shapes

User can find the area of the following shapes.

  • 1.Square
  • 2.Rectange
  • 3.Triangle
  • 4.Circle
  • 5.Parallelogram
  • 6.Trapezoid

Option 3 - Odd or Even

User can check whether the input number is odd or even.
